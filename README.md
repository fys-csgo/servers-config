### 管理员说明
- 配置文件将会通过github webhook推送.
- 管理员commit后将立即在服务器内生效.
- 请各个管理员不要频繁push.
  
  
### 注意事项
- 文本缩进统一为4空格缩进
- 文本换行符统一为LF
- 文本保存格式统一保存为UTF8-无BOM
- KeyValues文件禁止使用中文标点
- 禁止直接推送到master分支, 优先从PullRequest
